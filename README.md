# Solar System IIFE


## Description
Intro to augmented IIFE.  Getters and Setters for Planets, Active Spacecraft, Planets landed on, closest stars, and age of the solar system.

## How to run
In your terminal run:
```
http-server -p 9999
```
then navigate to 
```
http://localhost:9999
```
in your browser.
